User opens single page app https://studies.cs.helsinki.fi/exampleapp/spa


browser -> server: HTTP GET https://studies.cs.helsinki.fi/exampleapp/spa
server -> browser: spa (html file)
browser -> server: HTTP GET https://studies.cs.helsinki.fi/exampleapp/main.css
server -> browser: main.css
browser -> server: HTTP GET https://studies.cs.helsinki.fi/exampleapp/spa.js
server -> browser: spa.js

browser starts executing spa.js, which requests contents of data.json from https://studies.cs.helsinki.fi/exampleapp/data.json
and creates handlers

browser -> server: HTTP GET https://studies.cs.helsinki.fi/exampleapp/data.json
server -> browser: json data

browser renders notes
