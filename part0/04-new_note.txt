
User writes "new note" in the box and clicks "submit" button

browser -> server: HTTP POST https://studies.cs.helsinki.fi/exampleapp/new_note note="new note"
browser -> server: HTTP GET studies.cs.helsinki.fi/exampleapp/notes
server -> browser: notes (html file)
browser -> server: HTTP GET studies.cs.helsinki.fi/exampleapp/main.css
server -> browser: main.css
browser -> server: HTTP GET studies.cs.helsinki.fi/exampleapp/main.js
server -> browser: main.js


browser starts executing main.js, which requests data.json from studies.cs.helsinki.fi/exampleapp/data.json

browser -> server: HTTP GET studies.cs.helsinki.fi/exampleapp/data.json
server -> browser: json data

browser executes event handler and re-renders notes
